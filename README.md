# Meeting Manager Discord Bot

Tools and Languages
- Javascript
- Discord.js library
  
**Main Requirement**
- Report current meeting issues when user is online or join a voice channel (start meeting)
- Report Incoming (In-Progress Status), Urgent and Must have meeting issues
- Detect and report new meeting issues when user edited meeting spreadsheet
- Success and Failed status meeting issue should not be reported

Incoming Requirement
- Bot must be ready when any user is online
- Discord user is able to use command via text box to get bot to do specific things on server
- Discord user is able to add bot to private chat and command it to do specific things

Prerequisites
- Google Spreadsheet file for meeting issues
- Google Sheets API
- Discord server and its id
- Discord bot with its token
